## 验收操作建议

### 前置准备
1. 在全新或干净的 macOS 上运行 `curl -fsSL <install-url> | bash`
2. 准备一个 Telegram Bot Token（通过 @BotFather 创建）
3. 创建一个 Telegram 群组，将 Bot 加入并设为管理员

### 验收顺序
1. **#6 install.sh** → 确认安装流程无报错
2. **#7 /setup** → 在群组中执行 /setup，完成三步引导
3. **#1 /setup UI** → 同时观察引导界面的体验
4. **#2 /projects** → 执行 /projects，观察按钮布局
5. **#3 launchd** → 执行 `claude-pilot enable`，确认 plist 加载
6. **#4 崩溃通知** → 手动 `kill` Bot 进程，确认手机收到通知
7. **#5 自动恢复** → 确认 Bot 在 30 秒内自动重启
8. **#8 消息延迟** → 在 Claude 会话中观察消息推送延迟
9. **#9 并发安全** → 在同一话题中快速连续发送多条消息
10. **#10 TUI 审批** → 触发需要权限的操作，在手机上审批
11. **#11 对话流** → 观察 TG 话题中的消息格式
12. **#12 历史回显** → 执行 /resume，观察摘要内容
